## One CLI. All your registrars.

Manage your entire domain portfolio across multiple providers through a unified, scriptable terminal interface.

:::code-group

```bash [Unified List]
$ opendom domains --all
+------------------+-------------+---------------+-------------+
| Domain           | Provider    | Expires       | Auto-Renew  |
+------------------+-------------+---------------+-------------+
| my-project.com   | Cloudflare  | 2026-06-15    | ON          |
| startup.io       | Netim       | 2027-01-20    | OFF         |
| personal.me      | Porkbun     | 2026-11-05    | ON          |
+------------------+-------------+---------------+-------------+
```

:::

:::code-group
### Multi-Provider Sync
Manage domains across four major registrars (Netim, Cloudflare, Porkbun, Namecheap) from a single session.

### Identical Command Surface
Commands like `opendom dns set` work uniformly across all supported providers. No more learning vendor-specific APIs.

### Zero-Friction Operations
No popups, no cross-selling, and no distraction. Just fast, clean domain management for developers and AI agents.

### Predictable Safety
Built-in `--dry-run` and mandatory `--yes` flags ensure you never accidentally spend money or modify records.
:::
