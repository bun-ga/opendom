# @bun-ga/opendom

`opendom` is a CLI for multi-provider domain management from the terminal.

## Install

```bash
npm install -g @bun-ga/opendom
opendom --help
```

## Supported Providers

- Netim
- Cloudflare
- Porkbun
- Namecheap
